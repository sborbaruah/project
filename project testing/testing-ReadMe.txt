Testing Read Me file

Unit Tests 
- tests functions implemented in design.js file (quickforms/js/dom/form/design.js)
- implemented using QUnit framework , files in (test/unit tests/framework)
- tests correctness of output

Functional Tests
- tests written corresponding to each use case
- the basic tests written before implementation of each function
- implemented using Mocha framework, files in (test/functional tests/framework)


Acceptance Tests
- tests written corresponding to each use case 
- tests run after implementation of required functions complete
- assigns pass or fail (1 or 0) if a use case is completed successfully
- implemented usign QUnit framework, files in (test/acceptance tests/framework)
